# Financial_PowerBI_Project

<b>Project Overview</b>

This project focuses on building a comprehensive financial analytics dashboard to analyze business performance across revenue, profitability, customer behavior, operational efficiency, and risk exposure.

The objective was to provide stakeholders with clear, data-driven insights to improve profitability and decision-making.

<b>Business Objectives</b>

- Analyze total revenue growth over time
- Track EBITDA and profit contribution
- Monitor receivables aging and risk exposure
- Identify top-performing products and regions
- Evaluate operational efficiency

<b>Tools & Technologies Used</b>

- Power BI – Data modeling & visualization
- DAX – KPI calculations & advanced measures
- Power Query – Data cleaning & transformation
- Excel / CSV Dataset – Data source

<b>Data Analysis Approach</b>
- Step 1: Data Understanding
  - Explored dataset structure
  - Identified key financial columns (Revenue, COGS, Opex, Receivables, etc.)
- Step 2: Data Cleaning
  - Handled missing values
  - Standardized date formats
  - Created calculated columns for Aging Buckets
- Step 3: Data Modeling
  - Built relationships between:
  - Date table
  - Product table
  - Region table
  - Financial fact table

<b>Customer & Product Insights</b>

- Category Demand by Sales
- Top Performing Locations
- Top Shipping Efficiency by Location

<b>Challenges Faced</b>

- Circular dependency errors in calculated columns
- Handling missing financial values
- Fixing percentage calculation issues (values exceeding 100%)
- Ensuring accurate EBITDA computation
- Optimizing visuals to avoid clutter

<b>Key Insights</b>

- A small number of products contribute to a large portion of EBITDA.
- Revenue growth showed seasonal variation.
- High receivables in 60+ day buckets indicate potential cash flow risk.
- Certain regions outperform others in both revenue and efficiency.
- Cancellation rate directly impacts profit margins.

<b>Recommendations for Stakeholders</b>

- Focus on top EBITDA-driving products.
- Reduce exposure in 60+ days receivable buckets.
- Improve logistics performance in underperforming regions.
- Monitor cancellation trends monthly.
- Optimize product mix based on contribution analysis.

<b>Conclusion</b>

This dashboard provides a 360-degree financial view of the business, combining revenue performance, profitability drivers, operational metrics, and risk indicators in one interactive solution.

By leveraging Power BI and DAX, the project transforms raw financial data into actionable insights, enabling leadership to:
- Make data-driven decisions
- Improve cash flow management
- Focus on high-profit segments
- Reduce operational inefficiencies
- Strengthen long-term financial stability

Overall, this project demonstrates strong capabilities in data modeling, financial analytics, DAX calculations, and business storytelling through visualization.
